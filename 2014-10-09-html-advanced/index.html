Tabellen - Basis

Tabelle

  • <table> Kein Inhalt</table>
  • Umschließt die komplette Tabelle

Zeile

  • <tr> Kein Inhalt</tr>
  • Ein Element pro Zeile

Zelle

  • <td> Endlich Inhalt</td>
  • Viele Zellen pro Zeile

Tabellen - Im Detail

Tabellen Kopf

  • <thead> Kein Inhalt</thead>

Tabellenzelle im Kopf

  • <th> Kopfzelle</th>

Tabellen Inhalt

  • <tbody></tbody>

W3C Rerefenz

  • http://www.w3.org/TR/html5/tabular-data.html

Formulare

  • Grundelement jeder interaktiven Seite
  • Suchfeld, Login, Newsletter Signup, ...
  • Basis: <form>
  • Eingabe-Felder: <input>
  • Button: <button>Abschicken</button>

Spezielle Input-Typen

Texteingabe Felder

  • <input type=text>
  • Standard Eingabe Feld

Select Feld

  • <select></select>
  • Beinhaltet <option value="value1">Value 1</option>

Radio Buttons

  • <input type=radio value='red'>Rot

Checkboxen

  • <checkbox label='orange'></checkbox>

Spezielle Input Typen

<input type=?>

  • email
  • password
  • tel
  • number
  • date

Eine Seite mit Formulare


            <html>
              <head>...</head>
              <body>
              <form>
                <label>Name
                <input type=text/>
                <label>Email
                <input type=email>
                <label>Password
                <input type=password>
                <label>Number
                <input type=number>
                <label>Date
                <input type=date>
                <input type=radio value='red'/> Rot
                <input type=radio value='green'/> Grün
                <input type=radio value='green'/> Blau
                <input type=checkbox value='red'/> Rot
                <input type=checkbox value='green'/> Grün
                <input type=checkbox value='green'/> Blau
                <select>
                  <option>Milch
                  <option>Kaffee
                  <option>Tee
                </select>
                <button>Abschicken
              </form>
              </body>
            </html>

Externe Inhalte

  • In bestimmten Fällen möchte man Inhalte von anderen Webseiten auf seiner Seite einbinden
  • Hierfür existiert <iframe>
  • Häufiger Anwendungsfall: Social Media Content

Eine Seite mit Iframe


            <html>
              <head>...</head>
              <body>
                <h1>Die MI-Webseite</h1>
                <iframe src='http://www.google.de' width=1000 height=500>
                </iframe>
              </body>
            </html>
