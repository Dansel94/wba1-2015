Was ist ein Framework?

Ein Framework (englisch für Rahmenstruktur) ist ein Programmiergerüst, das in der Softwaretechnik, insbesondere im Rahmen der objektorientierten Softwareentwicklung sowie bei komponentenbasierten Entwicklungsansätzen, verwendet wird.

Quelle: http://de.wikipedia.org/wiki/Framework (22.10.2014)

Vorteile // Nachteile von Frameworks

Vorteile

  • Leistungsfähigkeit
  • Kosten
  • Unterstützung
  • Sicherheit und Robustheit

Nachteile

  • Sie lernen das Framework, nicht die Sprache
  • Einschränkung
  • Code ist öffentlich

Syntax

Document-Ready-Funktion

Eine Seite kann erst zuverlässig manipuliert werden, wenn das DOM geladen ist.

$( document ).ready(function() {
    console.log( "ready!" );
});

Oder in Kurzschreibweise:

$(function() {
    console.log( "ready!" );
});

Elemente selektieren

jQuery unterstützt die meisten CSS3-Selektoren

Selektion nach ID oder Klassen:

$( "#myId" );
$( ".myClass" );

Selektion nach Attributen:

$( "input[name='first_name']" );

Pseudo-Selektoren:

// das erste a-Element mit der Klasse "external"
$( "a.external:first" );

// alle ungeraden tr-Elemente
$( "tr:odd" );

// alle Div-Elemente die sichtbar sind
$( "div:visible" );

Auswahlen können in Variablen gespeichert werden:

var divs = $( "div" );

Zugriff auf Attribute

Attribute werden über die Methode .attr() gesetzt und abgerufen

Defintion von Attributen:

$( "a" ).attr( "href", "allMyHrefsAreTheSameNow.html" );

$( "a" ).attr({
    title: "all titles are the same too!",
    href: "somethingNew.html"
});

Abrufen von Attributen:

$( "a" ).attr( "href" );

Manipulation von Elementen

Setzt oder gibt…
.html() den Inhalt des HTML-Elements.
.text() den Inhalt als reinen Text. HTML wird nicht interpretiert.
.attr() den Wert von angegeben Attribut.
.width() die Breite in Pixeln des ersten Elements in der Auswahl.
.height() gibt die Höhe in Pixeln des ersten Elements in der Auswahl.
.val() den Wert eines Eingabe-Elements von einem Formular.
Demo - Unterschied zwischen .html() und .text()

Hinzufügen und Entfernen von Elementen

.append() Fügt Inhalt am Ende des ausgewählten Elements ein
.prepend() Fügt Inhalt am Anfang des ausgewählten Elements ein
.before() Fügt Inhalt vor dem ausgewähltem Element ein
.after() Fügt Inhalt hinter dem ausgewähltem Element ein
.remove() Löscht das ganze Element und die Kind-Elemente
.empty() Löscht nur die Kind-Elemente aus dem ausgewähltem Element
Demo - .append, .prepend, .before und .after

Event-Handling

JQuery ermöglicht die einfache Definition von ereignisgesteuerten Reaktionen auf HTML-Elemente

Definition direkt über eine Funktion:

$( "p" ).click(function() {
    console.log( "You clicked a paragraph!" );
});

Definition über die .on-Funktion:

// Equivalent event setup using the `.on()` method
$( "p" ).on( "click", function() {
    console.log( "click" );
});
jQuery - API-Beschreibung zu Events

Event-Handling

  • Jeder Event-Handler-Funktion wird ein Event-Objekt übergeben welches verschiedene Methoden und Eigenschaften enthält.
  • Über das Schlüsselwort this kann innerhalb der Event-Handler-Funktion auf das DOM-Element zugegriffen werden, welches das Ereignis ausgelöst hat
  • mit $(this) kann das DOM-Element in ein jQuery-Objekt umgewandelt werden.
